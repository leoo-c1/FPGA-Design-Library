# FPGA Design Library
A library of resuable Verilog modules and projects for FPGA development.

## Target Hardware
These modules are configured for the Intel Cyclone IV EP4CE6E22C8N FPGA on the RZ-EasyFPGA A2.2 / RZ-EP4CE6-WX board by Ruizhi Technology Co.\
\
